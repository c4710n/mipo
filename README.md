# mipi

[![code style: prettier](https://img.shields.io/badge/code_style-prettier-ff69b4.svg)]

> A opinionated tool that aim to convert Markdown to PDF.

> Based on [ReLaXed](https://github.com/RelaxedJS).

## Installation

```sh
npm install mipi
```

## Usage

```sh
mipi --help
```
